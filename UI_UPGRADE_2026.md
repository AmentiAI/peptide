## 🎨 Design Features

### Modern UI Trends (2026):
1. **Vibrant Gradients** - Purple/pink/blue spectrum
2. **Glassmorphism** - Frosted glass effects everywhere
3. **Bento Grids** - Card-based modular layouts
4. **Micro-animations** - Smooth hover states
5. **Depth with Shadows** - Layered shadow system
6. **Bold Typography** - Black weights, gradient text
7. **Icon-First Design** - Visual hierarchy
8. **Sticky Navigation** - Better UX
9. **Scroll Animations** - Progressive reveals
10. **Trust Indicators** - Badges, COAs, purity marks

### Color System:
- **Primary Gradients**:
  - Purple to Pink (667eea → 764ba2)
  - Blue to Cyan (4facfe → 00f2fe)
  - Green to Yellow (43e97b → 38f9d7)
  
- **Category Gradients**:
  - Healing: Blue → Cyan
  - Growth Hormone: Purple → Pink
  - Metabolic: Green → Yellow
  - Anti-Aging: Orange → Red
  - Immune: Teal → Cyan
  - Nootropic: Indigo → Violet

### Typography:
- **Font**: Inter (modern, clean)
- **Weights**: 400 (normal) → 900 (black)
- **Sizes**: Responsive clamp() values
- **Gradient Text**: Primary headings use gradient fills

### Spacing:
- **Base Unit**: 0.25rem (4px)
- **Scale**: xs, sm, md, lg, xl, 2xl, 3xl, 4xl, 5xl
- **Consistent**: All spacing uses scale

### Border Radius:
- **Small**: 0.5rem (8px)
- **Medium**: 0.75rem (12px)
- **Large**: 1rem (16px)
- **XL**: 1.5rem (24px)
- **2XL**: 2rem (32px)
- **Full**: 9999px (pills)
